How to configure

RetroArch-K

  • Online Updater > Update Core Info Files to fetch the available cores list. Use Core Downloader to download the desired core(s).
  • Settings > Input > RetroPad Binds > Port 1 Controls to configure the controls.
  • Kaillera Netplay in the main menu to load the Kaillera client. *

* The game(s) need to be loaded once locally, the client fetchs the available games from History.

Notes:

  • Some systems requires BIOS files to work, they go inside the system folder.
  • If there's no sound by default, DirectX Redist may need to be installed.

Notes (cores):

  • Flycast - Disable GPU multi-threading to prevent desyncs (F1 > Core Options > Performance > Thread Rendering OFF)

Limitations:

  • Systems that uses keyboards don't work online.

Hotkeys:

F2 - Save state
F4 - Load state
H  - Reset game

Game room:

!list    - Lists the users positions
!reset   - Restores default settings
!swap #  - Allows swapping inputs order between users
!take #  - Allows taking the inputs of other users
!version - Displays the client version from all users

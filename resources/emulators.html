RMG-K

How to configure it

  • Select ROM Directory... in the main menu to configure your games folder.
  • Settings > Input > Player 1 to configure the controls.
  • Netplay > Start Netplay to load the Kaillera client.

Suggestions:

  • Switch Input Plugin to raphnetraw (Settings > Settings > Plugins).

Snes9x++

How to configure it

  • Emulation > Settings... > Directory (Roms) to configure your games folder, restart Snes9x++ to apply changes.
  • Input > Input Configuration... to configure the controls.
  • Netplay > Kaillera Client... to load the Kaillera client.

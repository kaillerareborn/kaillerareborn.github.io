EmuLinker-K

How to run it

Linux

  • Install Java: * 
    $ sudo apt install openjdk-25-jdk      # Debian based distros
$ sudo dnf install java-25-openjdk     # Red Hat based distros
$ sudo pacman -S jdk-openjdk           # Arch based distros
$ sudo zypper install java-25-openjdk  # SUSE based distros
  • $ cd /path/server_folder
  • $ chmod +x *.sh
  • $ ./start-server.sh

macOS

  • Install Java.
  • (Rest of steps like Linux)

Windows

  • Install Java.
  • server.bat

Include the IP address of your server in the masterList.serverConnectAddress= value (emulinker.cfg) to make it appear in the Kaillera Master Lists.

* Change the number to 21 if the latest LTS version isn't yet available in your distro's repositories.

How to update it

  • Drag the new emulinker-k.jar file to the lib folder.
  • Replace the .bat and .sh files.
  • Restart EmuLinker.

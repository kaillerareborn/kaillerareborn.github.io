How to run ¹

EmuLinker-K

Linux (CLI)

  • Install Java: ²
    • # sudo apt install openjdk-25-jdk (Debian and derivatives)
    • # sudo dnf install java-25-openjdk (Red Hat and derivatives)
  • # cd server_folder and grant executable permissions: # chmod +x *.sh
  • # ./start-server.sh

Windows

  • Install Java (x64/.msi)
  • server.bat

¹ Include the IP address of your server in the masterList.serverConnectAddress= value (emulinker.cfg) to make it appear in the Kaillera Master Lists.

² Change the number to 21 (or 17) if the latest LTS version isn't yet available in your distro's repositories.

How to update

EmuLinker-K

  • Drag the new emulinker-k-*.jar file to the lib folder
  • Replace the .bat and .sh files
  • Restart EmuLinker to apply changes

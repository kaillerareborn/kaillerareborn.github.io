Kaillera offers the possibility to play retro games online using emulators with Kaillera support. It consists of a client and a server: the client functions are embedded in the emulators, while the server is a standalone application that needs to be run separately.

Kaillera Reborn is a community focused on maintaining the network by improving Kaillera software and hosting game servers in different locations. We also count with dedicated lists for EmuLinker-based servers and P2P game rooms.
